# Установка и настройка remnawave
Нам понадобится сервер от ``1:CPU 2:RAM 30:SSD``


# Покупа и настройка домена для панели:



| Касс     | Поддомен         | Домен           | IP / Значение        | Конечный результат          |
|:---------|:-----------------|:----------------|:---------------------|:----------------------------|
| **A**    | **@** (основной) | **mydomain.ru** | **185.114.xx.xx**    | mydomain.ru -> 185.114.xx.xx |
| **A**    | **www**          | mydomain.ru     | 185.114.xx.xx        | www.mydomain.ru -> 185.114.xx.xx |
| **A**    | **node**         | mydomain.ru     | **192.0.0.0 от сервера**      | node.mydomain.ru -> 192.168.0.1 |
| **CNAME**| **sub**          | mydomain.ru     |         | sub.mydomain.ru является алиасом |
| **CNAME**| **panel**        | mydomain.ru     |         | panel.mydomain.ru является алиасом |

Дальше идет установка скрипта.



## Начнем со скрипта от [eGames](https://github.com/eGamesAPI/remnawave-reverse-proxy)

### Обновления Ubuntu
```
apt update && apt upgrade -y
```

### Установка скрипта:
```
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```
##### Выбираем пункт:

1. Установить панель и ноду на один сервер.

- Указываем доманы которые мы создадали ранее

- Устанавливаем ssl выбрав вариант 1,2,3 какой удобно. [Если ошиблись просто начните все заново]

- Получаем данные от входа
